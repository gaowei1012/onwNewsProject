# onwNewsProject

### 安装运行

进入项目根目录 
```
cd project/
```

安装项目依赖

``` 
npm install
```

安装 `ios` 依赖

```
pod install 
```

运行
```
/* 保证本地装有 react native 开发环境 */
react-native run-ios 
```

## 配置第三方新闻接口
```
http://v.juhe.cn/toutiao/index
```
在聚合数据中申请，每日限额100条数据，仅供测试使用。

#### 解释
- 本项目新闻接口使用的是第三方接口，请不要用在商业上、传播。
